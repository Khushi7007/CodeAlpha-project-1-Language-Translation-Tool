# 🌍 AI Language Translation Tool

A Flask-based web application that translates text between multiple languages using Google Translate. Users can enter text, select the source and target languages, and instantly receive translated text through a simple and user-friendly interface.

## 🚀 Features

- 🌐 Translate text between multiple languages
- 🔍 Auto language detection
- 🎤 Voice input (Speech-to-Text)
- 🔊 Text-to-Speech
- 📋 Copy translated text
- 🔄 Swap source and target languages
- 🗑️ Clear input and output
- 💾 Download translated text as a TXT file
- 📱 Responsive and modern user interface

## 🛠️ Technologies Used

- Python
- Flask
- HTML5
- CSS3
- JavaScript
- Deep Translator (Google Translate)
- Browser Speech Recognition API
- Web Speech API

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/AI-Language-Translation-Tool.git
```

2. Navigate to the project folder

```bash
cd AI-Language-Translation-Tool
```

3. Install dependencies

```bash
pip install -r requirements.txt
```

4. Run the application

```bash
python3 app.py
```

5. Open your browser and visit

```
http://127.0.0.1:5000
```
